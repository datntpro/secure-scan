## Design Elements

### Color Coding
- 🟢 Green: Verified domains, good scores
- 🟡 Yellow: Warnings, medium issues
- 🔴 Red: Critical issues, unverified
- 🔵 Blue: Information, actions

### Risk Score Display
- A: Green with "Xuất sắc"
- B: Light green with "Tốt"
- C: Yellow with "Trung bình"
- D: Orange with "Kém"
- F: Red with "Nghiêm trọng"

### Interactive Elements
- Hover effects on cards
- Loading states for actions
- Real-time updates for scan progress
- Notification badges

### Responsive Design
- Mobile-first approach
- Collapsible sidebar on mobile
- Touch-friendly buttons
- Swipe gestures for cards
